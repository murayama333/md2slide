# フレックスボックスとは

+ FlexboxとはCSS3で導入された柔軟なレイアウトシステム
+ 横方向（または縦方向）に要素を並べることができる
+ 以下の2つの概念で成立する
  + **Flex Container**
  + **Flex Item**
