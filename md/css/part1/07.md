# コメントの記述方法

+ CSSのコメントは `/* */` で囲んで記述する
+ コメントはブラウザに無視される（表示に影響しない）
+ コードの説明や一時的な無効化に使用する

## サンプルコード

### part1.css

```css
p {
    /* 文字色を青にする */
    color: blue;
}
```

## ポイント

⚠️ HTMLのコメント `<!-- -->` と書き方が異なります。

CSSファイル内ではHTMLのコメントは使用できません。
