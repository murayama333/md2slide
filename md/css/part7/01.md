# positionプロパティ

+ `position` プロパティは要素の位置指定を制御する
+ デフォルト値は `static` で通常の文書フローに従って配置される
+ `position` プロパティに指定する代表的な値は以下のとおり
  + `static`
  + `relative`
  + `absolute`
  + `fixed`
  + `sticky`
