# HTMLの基本構造 - DOCTYPE宣言

+ **DOCTYPE宣言**はHTMLのバージョンをブラウザに伝える宣言
+ `<!DOCTYPE html>` はHTML5の文書型宣言
+ 2025年現在はHTML5を使用するのが一般的

## サンプルコード

### basic.html

```html
<!DOCTYPE html>
<html lang="ja">

<head>
    <meta charset="UTF-8">
    <title>タイトル</title>
</head>

<body>
    <h1>見出し</h1>
    <p>段落</p>
    <ul>
        <li>箇条書き1</li>
        <li>箇条書き2</li>
        <li>箇条書き3</li>
    </ul>
</body>

</html>
```
