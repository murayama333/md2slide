# HTML要素（Element）- 内容（Content）

![HTML要素（Element）- 内容（Content）-1](https://raw.githubusercontent.com/murayama333/md2slide/refs/heads/main/md/html/part1/img/08.png)

* **内容**とは開始タグと終了タグの間に挟まれるデータ
* **他のHTML要素を含むこともできる**
* **内容を持たない要素もある**（`<img>` `<br>` `<input>` など）
