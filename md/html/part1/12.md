# HTMLの基本構造 - `<head>`

+ `<head>` タグ内にはページに関する情報を記述する
+ 具体的にはHTML文書の文字コード、タイトルなど
+ CSSファイルやJavaScriptファイルの読み込みも `<head>` タグ内に記述することが多い

## サンプルコード

### basic.html

```html
<!DOCTYPE html>
<html lang="ja">

<head>
    <meta charset="UTF-8">
    <title>タイトル</title>
</head>

<body>
    <h1>見出し</h1>
    <p>段落</p>
    <ul>
        <li>箇条書き1</li>
        <li>箇条書き2</li>
        <li>箇条書き3</li>
    </ul>
</body>

</html>
```
