# スライドの作り方 - 見出し1 のみの場合

+ Markdownで記述したコンテンツが1ページで表示されます
+ 画像やコードブロックを追加できます
+ シンプルなページや、アジェンダページの作成に便利です

### slide-d.html

```html
<h1>SLIDE-D</h1>
<p>IT講師のための .md ファイルをスライドに変換するツール</p>
```
