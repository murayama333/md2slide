# スライドの作り方 - 見出し1 と 見出し2（※2つ） の場合

+ 見出し1 `#` 直下の内容が初期表示されます
+ 次に進む、と1つ目の見出し2 `##` の内容が画面右に縦長で表示されます
+ さらに次に進む、と2つ目の見出し2 `##` の内容が `ポップアップスライド` 表示されます

### .md ファイルの記述例

```

+ さらに次に進む、と2つ目の見出し2 `##` の内容が `ポップアップスライド` 表示されます

\#\# 1つ目の見出し2 画面右に表示される内容

+ サンプルコードのファイル名は `###` で書くとよいです
```
