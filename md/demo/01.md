# スライドの作り方 - 見出し1 のみの場合

+ Markdownで記述したコンテンツが1ページで表示されます
+ 画像やコードブロックを追加できます
+ シンプルなページや、アジェンダページの作成に便利です

### Hello.java

```java
public class Hello {
    public static void main(String[] arsgs) {
        System.out.println("Java");
    }
}
```
